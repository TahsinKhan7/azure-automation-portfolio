# KQL Queries

Azure Monitor and Log Analytics queries for platform monitoring, alerting and cost analysis.

| Query | Purpose |
|-------|---------|
| `pipeline-failure-alerts.kql` | Monitor CI/CD pipeline and ADF pipeline failures |
| `resource-health-dashboard.kql` | Platform health metrics across compute, storage and networking |
| `cost-anomaly-detection.kql` | Detect unusual spend patterns and resource provisioning spikes |

## Usage

These queries can be used in:
- Azure Monitor alert rules
- Log Analytics workbooks and dashboards
- Grafana via the Azure Monitor data source
- Azure DevOps pipeline monitoring widgets
